 * basic utilities:

   * BitPackedSequence
   * OffsetIterator

   * SizeTypeFor

   * AbstractFactory
   * Dispatcher
 * system utilities:
   * Descriptor
   * TempFile
 * threading:
   * ActiveObject
   * BasicLockable
   * CacheLine
   * Event
   * Fifo
   * JoiningThread
   * LockProxyProvider
   * WaitWrapper
